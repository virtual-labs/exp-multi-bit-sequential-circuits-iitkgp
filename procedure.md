## Procedure

 Please follow these steps to do the experiment.
 
 #### Part 1:
                         
 1.  At first apply high voltage to V<sub>cc</sub>.So that the "Clock Start" button
	for the clock pulse(<font style="text-decoration:overline">CP1</font>) will be enabled.<br/>
   
<img src="images/exp91.jpg" width="50%"/>
                                
<img src="images/exp92.jpg" width="50%"/>
                               
2.  Next, apply low voltage to the mode control input (S) and start the clock pulse(<font style="text-decoration:overline">CP1</font>).Now the serial input D<sub>s</sub> is enabled.The input at D<sub>s</sub> will be shifted right to the outputs Q<sub>0</sub>, Q<sub>1</sub>, Q<sub>2</sub>, Q<sub>3</sub> at negetive transition of <font style="text-decoration:overline">CP1</font>.All the parallel inputs P<sub>0</sub>, P<sub>1</sub>, P<sub>2</sub>, P<sub>3 </sub> are disabled 

<img src="images/exp93.jpg" width="50%"/>

<img src="images/exp95.jpg" width="50%"/>
						
3. Now apply high voltage to D<sub>s</sub> input and set no of clock pulses to 1.See that the input will be shifted to
	Q<sub>0</sub> output at negetive clock edge.<br/>
    
<img src="images/exp96.jpg" width="50%"/>

4.  Next, start the clock pulse again.See the input is now shifted to the output Q<sub>1</sub>. <br/>
    
<img src="images/exp97.jpg" width="50%"/>

5.  Again start the clock pulse and see the input is shifted to the output Q<sub>2</sub>. 

<img src="images/exp98.jpg" width="50%"/>
                               
6.  Start the clock pulse again.Now see that at fouth clock pulse input is shifted to the output Q<sub>3</sub>. 

<img src="images/exp99.jpg" width="50%"/>
						
7. Apply high voltage to the mode control input (s).Now the "Clock Start" button for the second clock pulse(<font style="text-decoration:overline">CP2</font>) is enabled.Parallel inputs P<sub>0</sub>, P<sub>1</sub>, P<sub>2</sub>, P<sub>3</sub> are enabled.These parallel inputs are directly loaded to the outputs Q<sub>0</sub>, Q<sub>1</sub>, Q<sub>2</sub>, Q<sub>3</sub> respectively.

<img src="images/exp910.jpg" width="50%" />
                               
                               
8. Start the clock pulse (<font style="text-decoration:overline">CP2</font>).After generation of some clock pulses stop the clock by clicking in "Clock Stop" button.<br/>
    
<img src="images/exp911.jpg" width="50%" />

9.  Now apply high voltage to P<sub>0</sub> input and set no of clock pulses to 1.See P<sub>0 </sub> input is directly loaded to the output Q<sub>0</sub>.

<img src="images/exp913.jpg" width="50%"/>
                              
10.  Next, apply high voltage to P<sub>1 </sub> and start the clock pulse.P<sub>1</sub> input is directly loaded to Q<sub>1</sub>. 

<img src="images/exp914.jpg" width="50%"/>

11. Next, apply high voltage to P<sub>2 </sub> and start the clock pulse.P<sub>2</sub> input is directly loaded to Q<sub>2</sub>. 

<img src="images/exp915.jpg" width="50%"/>
                               
12. Now apply high voltage to P<sub>3 </sub> and start the clock pulse.P<sub>3</sub> input is directly loaded to Q<sub>3</sub>. 

<img src="images/exp916.jpg" width="50%"/>
